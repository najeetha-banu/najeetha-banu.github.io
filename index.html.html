Tensile Creep Prediction — ML Project
Worked with tensile creep experimental data collected over 10+ years (10,702 rows). The goal was to predict material compliance behavior so that engineers wouldn't have to wait months for new lab results.
Applied Prony series curve fitting to extract compliance coefficients and relaxation times from the raw measurement data. After careful outlier filtering, achieved an R² of 0.9920 and reduced average AUC error to 0.12%.
Built an end-to-end XGBoost prediction pipeline with automated preprocessing, hypothesis testing (t-test, z-test), and confidence interval analysis. Learned a lot about validating models on real scientific data where trust in results really matters.

Deep Learning · Capstone
EMOCARE — Facial Emotion Recognition for Mental Health

My main capstone project. The idea came from noticing that people struggling mentally don't always express it — but emotions sometimes show in facial expressions. I trained an EfficientNetB0 model with a custom attention mechanism on FER2013 (seven emotion classes: angry, disgust, fear, happy, sad, surprise, neutral) and extended it beyond simple image classification into a fuller monitoring system.

Beyond the base model, I added GIF/video frame-by-frame prediction to track emotional change over time, text sentiment analysis with TextBlob, a behavioral questionnaire, a Bayesian Network using pgmpy for probabilistic reasoning, and a fairness check using Fairlearn. I also compared EfficientNet with MobileNet to understand tradeoffs, and tested the trained model on the CK+ dataset to check generalization.

83% accuracy on FER2013 · Tested on CK+ for generalization

Generative AI · RAG
InsightGPT — Document Q&A Chatbot

Built a RAG-based chatbot that lets users upload a document (PDF or text) and ask questions from it. The system splits the document into overlapping chunks, converts each chunk into embeddings using Hugging Face sentence-transformers, stores them in a FAISS vector index, and retrieves the most relevant chunks when a question is asked. FLAN-T5 then generates an answer from the retrieved context.

I also added a Gradio interface so it feels like a proper app, experimented with conversation memory so the bot can reference earlier questions, and saved the FAISS index to avoid rebuilding from scratch. I ran into hallucination issues where the model would answer even when the document didn't have the information — a reminder that RAG systems need good "I don't know" handling too.

ML · Optimization · Research
Ad Pricing & Bidding Optimization

This project turned into my IEEE publication. The problem: online advertisers compete in auctions for limited ad slots with limited budgets — how should they bid to maximize clicks efficiently?

I built a CTR prediction pipeline comparing Logistic Regression, Random Forest, and XGBoost (XGBoost performed best) using user, ad, contextual, and auction features. Then I connected predicted CTR to a convex optimization layer that allocates budget based on expected click value, and added a game-theoretic component to simulate competitive bidding between multiple agents.

Dynamic Advertisement Pricing and Bidding Optimization Using Machine Learning
Najeetha Banu N et al.  ·  IEEE Access, 2025

This paper addresses the real-time bidding problem in digital advertising — where advertisers must decide how much to bid for each ad slot, under budget constraints, while competing against other bidders. Rather than treating prediction and decision-making as separate problems, the paper proposes a unified pipeline that connects ML-based click-through rate prediction, convex budget optimization, and multi-agent auction simulation into one cohesive framework.

The system learns CTR from historical campaign data using XGBoost (benchmarked against Logistic Regression and Random Forest), feeds those estimates into a convex optimizer to allocate budget toward the highest-value opportunities, and validates the bidding strategy through a simulated auction environment where multiple advertisers compete simultaneously. The result bridges supervised learning, mathematical optimization, and game theory in a way that reflects how real advertising platforms actually work.
